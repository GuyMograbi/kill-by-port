# Installation

```
npm install -S GuyMograbi/kill-by-port
```

or globally for command line

```
npm install -g GuyMograbi/kill-by-port
```

# Usage

```
require('kill-by-port').killByPort(8000)
```

or by command line

```
kill-by-port --port 8000
```
